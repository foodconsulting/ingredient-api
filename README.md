# Ingredient API

SpringBoot simple CRUD API made for ingredients.

## Technologies

* Java 17
* SpringBoot v3.3.3
* Maven

## Dependencies

* SpringBoot DevTools
* Spring Web
* Spring Data JPA
* Postgresql
* Lombok

## Testing

* SpringBoot Test
* JUnit5
* H2
